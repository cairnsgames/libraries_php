Currency Conversion API

The convert.php file provides an API endpoint for converting currency amounts between different currencies. It uses exchange rates stored in the database to perform the conversion.

Endpoint

https://cairnsgames.co.za/php/exchange/convert.php

Parameters

  • from (required): The source currency code (e.g., USD).
  • to (required): The target currency code (e.g., ZAR).
  • amount (required): The amount to convert (e.g., 1000).
  • date (optional): The date for historical exchange rates (format: YYYY-MM-DD). If not provided, the latest available date is used.

Example Usage

To convert 1000 USD to ZAR:

https://cairnsgames.co.za/php/exchange/convert.php?from=USD&to=ZAR&amount=1000

Response

The API returns a JSON response:

{
    "success": true,
    "from": "USD",
    "to": "ZAR",
    "amount": 1000,
    "converted": 18500.123456,
    "date": "2025-06-20"
}

Error Handling

If required parameters are missing or an error occurs, the API returns an error response:

{
    "success": false,
    "error": "Missing required parameters: from, to, amount"
}

Notes

  • Ensure the from and to currency codes are valid.
  • The amount should be a positive number.
  • Historical rates are only available if stored in the database.
