Tenant API Documentation

Endpoints

GET /gettenant

Description: Retrieves tenant information based on the application ID.

Parameters:

  • appid: Application ID (UUID)

GET /getsettings

Description: Retrieves application settings based on the application ID and domain. The domain field is the hostname of the source application, allowing different settings based on the server the front end is deployed to.

Parameters:

  • appid: Application ID (UUID)
  • domain: Client domain (optional)

GET /getsecret

Description: Retrieves application secrets based on the application ID and domain. The domain field is the hostname of the source application, allowing different settings based on the server the front end is deployed to.

Parameters:

  • appid: Application ID (UUID)
  • domain: Client domain (optional)

GET /api.php/params

Description: Retrieves application properties based on the application ID.

Parameters:

  • appid: Application ID (UUID)

GET /api.php/tenant

Description: Retrieves tenant details such as ID, UUID, name, and description.

Parameters:

  • appid: Application ID (UUID)
