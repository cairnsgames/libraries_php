Tenant API Documentation

The tenant is also referred to as an app_id. When calling api.php, the app_id must be sent.

Endpoints

Get Tenant Details

URL: https://cairnsgames.co.za/php/tenant/api.php/tenant

HTTP headers

  • app_id: Tenant-specific value sent as a header. (REQUIRED)

Response:

{
  "id": 47,
  "uuid": "83b8771c-07f9-11f0-9750-1a220d8ac2c9",
  "name": "AccessElf",
  "description": null
}

Get System Settings

URL: https://cairnsgames.co.za/php/tenant/getsettings.php

HTTP headers

  • app_id: Tenant-specific value sent as a header. (REQUIRED)
